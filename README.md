Football Vision – AI Player Tracking System

An AI-based computer vision system for real-time player and ball tracking in football match footage.

Features
- YOLOv8 for object detection
- Multi-object tracking of players and ball
- Team classification using color clustering
- OpenCV-based visual annotations
- Generates annotated video output with player analytics

Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
